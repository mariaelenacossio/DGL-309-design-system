Spacing

Different spacing configurations affect the overall layout and visual appearance of the components.

Theres some set-up classes to ensure the margins and spacing in each container will be responsive and adjust to every screen.

Spacing

This is a .container class that it will keep the content inside the .div aligned and centered in the page which in this case is the empty space between the component and its content.

  • Padding-top its set to 1rem
  • Padding-left its calculated by a formula calc(1.5rem * 0.5)
  • Padding-right its calculated by a formula calc(1.5rem * 0.5)
  • The padding bottom is leaved unset only to define it once the content is in the site.
